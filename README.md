# DSIP Smule Bot - Полнофункциональный Telegram бот

Мощный Telegram бот для сообщества DSIP Smule с множеством функций для взаимодействия и развлечения.

## 🚀 Возможности

### **Основные функции:**
- ✅ Работает в каналах и группах
- ✅ **Отвечает в личных сообщениях** с умными ответами
- ✅ **Модульная структура** - команды в отдельных файлах
- ✅ Логирование всех действий
- ✅ Простая настройка через переменные окружения

### **Развлечения:**
- 🎲 **Игра "Поле чудес"** с голосовыми подсказками
- 🎰 **Казино** - анимированные слоты с шансом выигрыша
- 🎵 **Случайные песни** с Last.fm API
- 🐱 **Случайные котики** 
- 😂 **Случайные мемы** с Imgflip API
- 🔮 **Вокальные предсказания**
- 🤖 **AI-персонаж** с голосовыми ответами

### **Социальные функции:**
- 🏆 **Зал славы/позора** с номинациями и голосованием
- 🔥 **Критика исполнений** (/roast)
- ✅ **Подтверждение исполнений** (/proof)
- 🎤 **Оценка треков** через ссылки Smule

### **Игровые механики:**
- ⏰ **Ограничения по времени** для предотвращения спама
- 🎯 **Интерактивные игры** с сохранением состояния
- 🎵 **Голосовые подсказки** для игр
- 📊 **Система голосования** и номинаций

## Установка

### 1. Клонирование и настройка

```bash
# Перейдите в директорию проекта
cd smule-bot

# Активируйте виртуальное окружение
source venv/bin/activate

# Установите зависимости (если еще не установлены)
pip install -r requirements.txt
```

### 2. Создание бота в Telegram

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Сохраните полученный токен

### 3. Настройка переменных окружения

Скопируйте файл `env.example` в `.env` и заполните его:

```bash
cp env.example .env
```

Отредактируйте файл `.env`:

```env
# Telegram Bot Token (получите у @BotFather)
BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

# ID канала (начинается с -100)
CHANNEL_ID=-1001234567890

# ID группы (начинается с -100)  
GROUP_ID=-1001234567890

# Hetzner Object Storage credentials for NASSAL2026 registrations
S3_ACCESS_KEY=your_object_storage_access_key
S3_SECRET_KEY=your_object_storage_secret_key
```

### 4. Получение ID канала и группы

#### Для канала:
1. Добавьте бота в канал как администратора
2. Отправьте любое сообщение в канал
3. Перейдите по ссылке: `https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates`
4. Найдите `"chat":{"id":-1001234567890}` - это ID канала

#### Для группы:
1. Добавьте бота в группу как администратора
2. Отправьте команду `/start` в группе
3. Перейдите по ссылке: `https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates`
4. Найдите `"chat":{"id":-1001234567890}` - это ID группы

## 📋 Команды бота

### **Основные команды:**
- `/start` - запуск бота и приветствие
- `/help` - список всех доступных команд
- `/hello` - простое приветствие
- `/info` - информация о боте и чате

### **Развлечения:**
- `/prediction` - получить вокальное предсказание
- `/random` - случайная русская песня с Last.fm
- `/cat` - случайное изображение котика
- `/meme` - случайный мем
- `/casino` - игра в слоты с анимацией
- `/pole` - игра "Поле чудес" с голосовыми подсказками
- `/ask [вопрос]` - задать вопрос AI-персонажу с голосовым ответом

### **Зал славы/позора:**
- `/hall [legend/cringe] [имя]` - номинировать пользователя
- `/halllist` - посмотреть списки зала славы/позора
- `/vote [legend/cringe] [имя]` - проголосовать за номинанта

### **Критика и подтверждение:**
- `/roast` - получить критику исполнения (ответом на сообщение)
- `/proof` - подтвердить исполнение (ответом на сообщение)

## Личные сообщения

Бот **умно отвечает** в личных сообщениях на основе ключевых слов:

- **Приветствие**: "привет", "здравствуй", "hi", "hello"
- **Вопросы**: "как дела", "что нового", "как ты"
- **Благодарность**: "спасибо", "благодарю", "thanks"
- **Прощание**: "пока", "до свидания", "bye"
- **Помощь**: "помощь", "help", "что умеешь"
- **Время**: "время", "дата", "день"
- **О боте**: "бот", "robot", "искусственный"

Если ключевые слова не найдены, бот отвечает случайным дружелюбным сообщением.

## Разработка

### Добавление новых команд

Теперь команды вынесены в отдельные файлы! Чтобы добавить новую команду:

1. **Создайте новый файл** в папке `commands/` (например, `commands/mycommand.py`):
```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
Команда /mycommand
"""

from telegram import Update
from telegram.ext import ContextTypes

async def mycommand_command(update: Update, context: ContextTypes.DEFAULT_TYPE):
    """Обработчик команды /mycommand"""
    await update.message.reply_text("Ответ на команду!")
```

2. **Импортируйте команду** в `bot.py`:
```python
from commands.mycommand import mycommand_command
```

3. **Добавьте обработчик** в метод `setup_handlers`:
```python
self.application.add_handler(CommandHandler("mycommand", mycommand_command))
```

### Добавление новых ответов для личных сообщений

Отредактируйте файл `commands/message_handler.py` и добавьте новые условия в функцию `handle_personal_message`.

### Логирование

Бот автоматически логирует все действия. Логи выводятся в консоль с указанием времени, уровня и сообщения.

## Устранение неполадок

### Бот не отвечает на команды

1. Убедитесь, что токен бота правильный
2. Проверьте, что бот добавлен как администратор в канал/группу
3. Убедитесь, что файл `.env` существует и содержит правильные значения

### Ошибка "BOT_TOKEN не найден"

1. Создайте файл `.env` в корне проекта
2. Добавьте в него строку: `BOT_TOKEN=ваш_токен_здесь`
3. Перезапустите бота
